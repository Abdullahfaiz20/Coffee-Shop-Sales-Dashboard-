# ☕ Coffee Shop Sales Dashboard | Microsoft Excel

An interactive Microsoft Excel dashboard designed to analyze coffee shop sales performance, customer purchasing behavior, and business KPIs. This project demonstrates data analysis, dashboard development, and business intelligence skills using Excel.

---

## 📌 Project Overview

This dashboard provides valuable insights into coffee shop operations by analyzing sales transactions across different store locations, product categories, and time periods. It enables business stakeholders to monitor performance and make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Track customer footfall and purchasing trends.
- Identify top-selling products and categories.
- Compare store performance across locations.
- Visualize sales patterns by day and hour.

---

## 📊 Dashboard Highlights
